Projetos

Projeto 1

API WhatsApp

API REST desenvolvida para simular funcionalidades de um sistema de mensagens inspirado no WhatsApp. Permite acessar dados de usuários, contatos e conversas, incluindo busca por palavras-chave dentro das mensagens. O projeto foi focado na organização de rotas, estruturação de endpoints REST e manipulação de dados em JSON, reforçando conceitos de back-end com Node.js.

Node.js

Projeto 2

Site da Rifa

Site desenvolvido em equipe com o objetivo de divulgar e organizar uma rifa para arrecadação da turma. A aplicação apresenta os prêmios, regras e informações da ação de forma clara e acessível. O projeto reforçou habilidades de trabalho em grupo, estruturação de layout com HTML e estilização com CSS.

HTML CSS JavaScript
🔗 Ver site 📂 GitHub

Projeto 3

API de Estados e Cidades

API REST criada para disponibilizar dados de estados e cidades do Brasil de forma estruturada. Permite consultas organizadas, facilitando o consumo dessas informações em outras aplicações. O projeto foi desenvolvido para praticar criação de endpoints, organização de rotas e manipulação de dados em JSON utilizando Node.js.

Node.js

Projeto 4

Landing Page (Estacionamento)

Landing page desenvolvida para apresentar um sistema de gerenciamento de estacionamento, criada como parte de um projeto integrador do SENAI. A página tem foco em divulgação do software, destacando funcionalidades e benefícios. O projeto reforçou conceitos de layout, responsividade e construção de interfaces com HTML, CSS e JavaScript.
